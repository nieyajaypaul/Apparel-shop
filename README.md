Project Title

Online Apparel Shop

Project Description

The Online Apparel Shop is a simple web-based e-commerce application developed using HTML, CSS, and JavaScript. The purpose of this project is to provide users with an easy and convenient way to browse clothing products and purchase them online. The website displays different apparel items with their prices and allows users to add products to a shopping cart.

The system includes a home page that introduces the website and provides navigation to different sections such as the products page and the cart page. On the products page, users can view available clothing items, select the quantity they wish to purchase, and add them to their cart. The cart page displays all the selected products along with their price, quantity, and the total amount to be paid.

The project also includes a basic checkout feature where users can proceed to payment and choose from multiple payment options such as UPI, debit or credit card, net banking, or cash on delivery. This project demonstrates the basic functionality of an online shopping website and helps in understanding how e-commerce platforms manage products, carts, and payments.

Technologies Used

HTML for structuring the web pages

CSS for styling and designing the website layout

JavaScript for interactive features such as adding items to the cart and calculating the total amount

Features

User-friendly home page with navigation menu

Product listing with images and prices

Quantity selection for products

Add to cart functionality

Shopping cart with total price calculation

Checkout and payment options

Responsive and simple interface for easy use
